## 简介

这是任霏的个人网站（[www.renfei.net](https://www.renfei.net)），一直保持一年一次大更新，所以这个版本是为2022年准备的。

代号 renfeid 是因为参考了一部分 DDD 领域设计，从之前的贫血模型到现在的充血模型的应用；并且 Linux 中的服务都已 d 结尾来表示 daemon，伴随着系统的启动而启动，系统的关闭而结束，在后台等待执行的进程。
